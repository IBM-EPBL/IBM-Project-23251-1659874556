# Inventory-Management

# Languages Used:
- Python
- JQuery
- HTML

# Databases Used:
- MySQL

# Requirements
- Python Packages:
  - WTForms
  - SQLAlchemy
  - MySQL
- Server(Localhost)
- MySQL Server

# To Run the app
- Install docker and docker compose
- Build docker image using `docker compose build`
- Run `docker compose up -d`
- Access application at http://localhost:8000
